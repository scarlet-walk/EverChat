 شات + خرائط + مساعد GPT أونلاين وأوفلاين

🎯 الفكرة العامة

تطبيق واحد يجمع:

• شات أونلاين وأوفلاين (عبر الإنترنت أو البلوتوث/Wi‑Fi Direct).
• خرائط أونلاين وأوفلاين (تحميل مناطق للعمل بدون إنترنت).
• مساعد GPT ذكي يعمل أونلاين وأوفلاين، مدمج مع الشات والخرائط.


---

🗺️ تدفق المستخدم (User Flow)

1. الشاشة الرئيسية

• شريط تنقل سفلي:1. Chats 💬
‏2. Map 🗺️
‏3. Assistant 🤖



---

2. قسم المحادثات (Chats)

• أونلاين: محادثات عبر الإنترنت.
• أوفلاين: محادثات نصية بين الأجهزة القريبة.
• دمج GPT: كتابة `@GPT` داخل أي محادثة لاستدعاء المساعد.• أونلاين → GPT عبر API.
• أوفلاين → GPT محلي مصغّر.



---

3. قسم الخرائط (Map)

• عرض الخريطة أونلاين أو أوفلاين.
• تحميل مناطق محددة للعمل بدون إنترنت.
• مشاركة الموقع كنقطة دبوس في الشات.
• أيقونة GPT على الخريطة لطرح أسئلة عن الأماكن أو الطرق.


---

4. قسم المساعد (Assistant)

• واجهة شات خاصة بـ GPT.
• أوضاع المساعد:1. مساعد عام.
2. مساعد سفر.
3. مساعد طوارئ.

• يعمل أونلاين وأوفلاين.


---

5. مميزات إضافية

• زر SOS لإرسال استغاثة + الموقع للأجهزة القريبة.
• دليل طوارئ محلي (أرقام المستشفيات، الشرطة، الصيدليات).
• مزامنة تلقائية للرسائل والمواقع عند عودة الإنترنت.


---

📋 التصميم النصي للشاشات (Wireframe)

شاشة البداية

• شعار التطبيق + شريط تحميل.


شاشة تسجيل الدخول

• بريد/جوال + كلمة مرور.
• أزرار: تسجيل الدخول، إنشاء حساب، دخول كضيف.
• خيارات: Google / Apple.


الشاشة الرئيسية

• شريط تنقل سفلي (Chats / Map / Assistant).
• شريط علوي (إعدادات / بحث).


شاشة المحادثة

• قائمة المحادثات.
• زر بدء محادثة جديدة.
• داخل المحادثة: صندوق كتابة، زر إرسال، زر مرفقات، دعم `@GPT`.


شاشة الخريطة

• عرض الخريطة.
• زر تحديد الموقع.
• زر تحميل/إدارة الخرائط.
• أيقونة GPT.


شاشة المساعد

• واجهة شات مع GPT.
• اختيار وضع المساعد.
• مؤشر أونلاين/أوفلاين.


شاشة الإعدادات

• تعديل الملف الشخصي.
• إدارة الخرائط.
• إعدادات GPT.
• الخصوصية.
• تسجيل الخروج.
